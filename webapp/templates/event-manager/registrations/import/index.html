TN importieren

Sie können Teilnehmende aus einer Tabellendatei importieren.

Für den Import muss die Tabellendatei in das Format CSV überführt werden.

Alle üblichen Tabellenkalkulationsprogramme bieten die Möglichkeit, eine Tabellendatei im Format .CSV abzuspeichern. Sie können die Warnungen, dass einige Funktionen ggf. nicht übertragen werden, für die Zwecke des Imports ignorieren.
